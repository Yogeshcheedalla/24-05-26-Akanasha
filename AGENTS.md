## graphify

This project has a navigable knowledge graph at graphify-out/.

Rules:
- Before answering architecture or codebase questions, read graphify-out/GRAPH_REPORT.md for god nodes and community structure.
- Utilize the knowledge graph structure to navigate complex dependencies instead of relying solely on `grep`.
- After modifying core logic, run `graphify update .` to keep the graph's structural nodes current.
